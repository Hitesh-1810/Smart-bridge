# MedSafe AI 🛡️
AI-Driven Medical Safety Assistant powered by Google Gemini.

## Features
- Symptom analysis with possible condition suggestions
- Drug interaction warnings
- First aid guidance
- Emergency red-flag detection
- Conversational memory (last 6 messages)

## Setup

1. Install dependencies:
   ```
   pip install -r requirement.txt
   ```

2. Add your Gemini API key to `.env`:
   ```
   GEMINI_API_KEY=your_key_here
   ```
   Get a free key at: https://aistudio.google.com/app/apikey

3. Run the app:
   ```
   python app.py
   ```

4. Open http://localhost:5000

## Disclaimer
MedSafe AI is for informational purposes only and is not a substitute for professional medical advice.
